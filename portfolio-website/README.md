# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript. This project showcases modern web development skills with a clean, professional design.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Smooth Navigation**: Smooth scrolling and active section highlighting
- **Interactive Elements**: Animated cards, hover effects, and transitions
- **Mobile Menu**: Hamburger menu for mobile devices
- **Multiple Sections**:
  - Home/Hero section with call-to-action
  - About section with profile information
  - Skills section showcasing technical abilities
  - Projects section displaying portfolio work
  - Contact section with form and social links

## Technologies Used

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with Flexbox, Grid, and CSS variables
  - Responsive design with media queries
  - CSS animations and transitions
  - Mobile-first approach
- **JavaScript**: Interactive functionality
  - Mobile menu toggle
  - Smooth scrolling navigation
  - Form handling
  - Scroll-based animations
  - Intersection Observer API

## How to Use

1. Open `index.html` in a web browser
2. Navigate through sections using the navigation menu
3. View the responsive design by resizing your browser window
4. Test the contact form (currently shows an alert message)

## Customization

To personalize this portfolio:

1. **Update Personal Information**:
   - Replace "John Doe" with your name in `index.html`
   - Update the hero subtitle with your title/role
   - Modify the About section content

2. **Add Your Projects**:
   - Edit the Projects section in `index.html`
   - Add your project titles, descriptions, and links
   - Update technology tags

3. **Update Skills**:
   - Modify the Skills section with your technologies
   - Adjust skill icons and descriptions

4. **Contact Information**:
   - Update email, phone, and location in the Contact section
   - Add your social media links (GitHub, LinkedIn, Twitter)

5. **Styling**:
   - Customize colors in the `:root` CSS variables in `styles.css`
   - Adjust spacing, fonts, or animations as needed
